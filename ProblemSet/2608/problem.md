# 递归求解函数
Time Limit: 1 Sec  Memory Limit: 2 MB


## Description
递归求解函数f(x,n)


-----------------------------------------------------------------------------

Invalid Word（禁用单词）错误：在解决这个题目时，某些关键词是不允许被使用的。如果提交的程序中包含了下列的关键词之一，就会产生这个错误。

被禁用的关键字：for, while, do, break, continue, goto。



## Input
第一个数是实数x，第二个数是整数n。

## Output
函数值，保留两位小数


## Sample Input
```
1 2

```
## Sample Output
```
0.40

```

# 有多少个素数
Time Limit: 2 Sec  Memory Limit: 128 MB


## Description
求区间m～n之间的素数，其中m,n为整数，且0<=m<=n<=10000000。


## Input
输入多行，到EOF结束。每行为一个测试用例，两个整数m和n。总测试用例数不超过1000个。


## Output
输出每行与输入对应，为m～n之间素数的个数。


## Sample Input
```
1 2
2 3
8 10
1 11
50 100
900 1000
499900 500000

```
## Sample Output
```
1
2
0
5
10
14
7

```

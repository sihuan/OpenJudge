# 最大跨度值
Time Limit: 1 Sec  Memory Limit: 2 MB


## Description
         给出若干整数，请计算它们的最大跨度。最大跨度值 = 最大值 – 最小值。


## Input
         输入为若干整数，至EOF结束，两两整数之间用空格分开。


## Output
         输出为一行，表示最大跨度值。格式见样例。


## Sample Input
```
3 0 8 7 5 9
```
## Sample Output
```
The maximum stretch is 9.

```

# 求下一天
Time Limit: 1 Sec  Memory Limit: 2 MB


## Description
         给出一个日期，输出下一天的日期。


## Input
         输入若干日期至EOF结束，格式为y-m-d，其中y、m、d是三个正整数表示年月日，均为合法日期，其中y的输入范围是1000～9999年。


## Output
         输出每个日期下一天的日期。


## Sample Input
```
2019-07-18
2022-01-01
2011-12-30
2033-11-30

```
## Sample Output
```
2019-07-19
2022-01-02
2011-12-31
2033-12-01

```

# 尼科梅彻斯定理
Time Limit: 1 Sec  Memory Limit: 2 MB


## Description
         任何一个正整数n的立方都可以唯一地写成n个相邻奇数之和。这就是尼科梅彻斯定理。例如，
         13 = 1
         23 = 8 = 3 + 5
         33 = 27 = 7 + 9 + 11
         43 = 64 = 13 + 15 + 17 + 19
现在你编程序计算n3是哪些相邻奇数之和。


## Input
         输入一个正整数n，且n<1291。


## Output
         输出为n个相邻整数，它们的和为n3。


## Sample Input
```
5
```
## Sample Output
```
21
23
25
27
29

```
